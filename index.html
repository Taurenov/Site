2023 — н.в. 3 года

Программист · многосервисное приложение для инвестиций

  • Backend (Go): REST API и WebSocket для котировок, авторизация, PostgreSQL, миграции, конфигурация через переменные окружения.
  • Engine (Rust): сервис финансовых расчётов (прогноз доходности, метрики) с gRPC/HTTP-интерфейсом.
  • Desktop (Tauri + React): управление портфелем, локализация ru/en, сборка под Windows.
  • Инфраструктура: Docker Compose — отдельные контейнеры для backend, engine, БД и Redis.

Pet-проекты: Go API-сервисы, production-style scraping; C/C++ — инструменты с DLL и Lua-скриптами.

Обо мне

Backend-разработчик на Golang, 24 года — и за пределами IDE я в Азероте играю шаманом: баланс стихий, тотемы и синергия командной игры отражаются в том, как я проектирую сервисы. Стек: Go, Rust, PostgreSQL, Redis, Docker, Tauri, C/C++.

Разрабатываю многосервисное приложение для инвестиций: десктоп-клиент, backend и отдельный Rust-engine для финансовых расчётов. Люблю продумывать систему целиком — от доменной модели и API до сборки и деплоя.

Ищу команду, где смогу развиваться как Golang backend-разработчик уровня middle: нагруженные проекты, ответственность за архитектуру и качество кода. Формат работы — удалённо.
