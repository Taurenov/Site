Как подключить свой фон (как на референсе Thorgal)
================================================

1. Подготовьте одно изображение и сохраните в эту папку как:
   hero-bg.jpg   (или hero-bg.webp — тогда в css/styles.css замените расширение)

2. Рекомендуемые параметры:
   • Размер: 2560×1440 или минимум 1920×1080
   • Формат: WebP (лучше) или JPG, качество 80–90%
   • Стиль: иллюстрация / цифровая живопись (НЕ скриншот UI)
   • Сюжет для шамана WoW: портал или тотем по центру, парящие камни,
     грозовое небо, тёплая земля внизу (как на референсе с воротами)
   • По центру — чуть темнее/пустее, чтобы читался белый заголовок
   • Края можно темнее — текст в углах не планируется

3. Временно стоит hero-bg.jpg с референса — замените файл своей картинкой
   с тем же именем, обновите страницу (Ctrl+F5).

4. Опционально — отдельный фон для мобильных (вертикальный кроп):
   hero-bg-mobile.jpg (подключение в CSS по желанию, напишите — добавлю)
